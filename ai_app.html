Building a full-scale sports simulator in a single HTML file is a bit like trying to fit a stadium into a lunchbox, but we can certainly get the "Google Doodle" vibe going. This app uses a **2D Canvas engine** to simulate a cricket match. It features a timing-based hitting mechanic, a "bowler" with varying speeds, and a physics-based scoring system. ### The Game: "Sticky Wicket" * **How to Play:** Wait for the ball to enter the hitting zone (the transparent circle) and **Click or press Spacebar** to swing. * **Timing is Key:** Hit it early to pull the ball left; hit it late to push it right. * **The Goal:** Score as many runs as possible before you lose 3 wickets. --- ```html Minimalist Cricket Pro
Score: 0
Wickets: 0/3
OUT!
```
